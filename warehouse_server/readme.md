## hi
Структура:
```
warehouse_server/
├── warehouse-app/
├── warehouse-server/
├── docker-compose.yml
├── initdb.js
├── README.md
└── images/
```

Для запуска проекта выполнить команду в корневой папке:
```
docker-compose up
```

Клиент доступен по:
http://localhost:5173

Сервер доступен по:
http://localhost:5005

БД доступна по:
http://localhost:27017
___
#### Использую
* **Express.js**
* **MongoDB**
* **Mongoose**
* **Postman**
___
